Project Notes

Expanded descriptions for each project.

Project 01

Mess App

This project is designed to reduce food wastage by connecting leftover food from messes or hotels with NGOs or other collectors who can pick it up in time. The workflow focuses on item name, quantity, and location so the response can be fast and practical.

Problem: leftover food waste Users: mess owners and collectors Goal: faster pickup coordination

Project 02

Smart Moisture Gesture Model for DEE

This project is a gesture-controlled smart moisture management system built for digital electronics. It uses a soil moisture sensor, IR gesture sensor, 555 timer, 7408 AND gate, 7473 JK flip-flop, relay, and water pump to control irrigation with both automatic and gesture-based logic.

Automatic mode: dry soil starts pump Gesture mode: wave once ON, wave again OFF Smart logic: pump works only when soil is dry

Project 03

AI Drowsiness Detection

This driver-safety project uses Python with a camera and face detection to identify whether a driver is becoming sleepy. When drowsiness is detected, the system gives an alert so that accidents can be avoided.

Python implementation Camera-based monitoring Alert for accident prevention
